# kr_utils

there are 2 scripts
cd to cloned repo directory and run:

on windws:
```bash
py 2_1.py
py 2_2.py
```

on unix:

```bash
python 2_1.py
python 2_2.py
```

note that second one allows `--verbose` and `-v` flags to show how set __h__ is generated
